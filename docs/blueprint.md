# **App Name**: FinanciMate

## Core Features:

- Quick Onboarding: Collect essential user data (income, expenses, goals) through a streamlined onboarding process.
- Surplus Calculation: Calculate the user's monthly surplus (income minus expenses).
- Goal Definition: Allow users to define financial goals (amount, optional target date, urgency).
- Emergency Fund Assessment: Assess the user's emergency fund status and highlight potential shortfalls.
- Prioritization Engine: Prioritize goals based on urgency and emergency fund status.
- Personalized Financial Tool: The system uses the reasoning to recommend how the user should contribute to reach its financial goals.
- Reporting & Export: Generate an easily understandable report (PDF/printable view) that includes applied rules.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and stability.
- Background color: Very light blue (#E3F2FD), a desaturated version of the primary to ensure a calm user experience.
- Accent color: Muted green (#81C784), analogous to blue, to highlight progress and success.
- Body and headline font: 'PT Sans' for its modern look and slight warmth.
- Use clean, minimalist icons to represent financial concepts (money, savings, debt, etc.).
- Subtle transitions and animations to guide users through the planning process and display results.
